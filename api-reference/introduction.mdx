## Overview

This API supports email verification and task status checks. Submit an email for verification and use the returned task ID to poll for results.

## Base URL

All requests use the base URL from your environment (for example, `http://sandbox.mintlify.com` for sandbox).

## Authentication

All endpoints require an API key. Send it in the `x-api-key` header with every request.

```json
"security": [
  {
    "apiKeyAuth": []
  }
]
```

You need a valid API key and an active subscription. Invalid or missing keys return `401`; expired or unpaid subscriptions return `402`.
